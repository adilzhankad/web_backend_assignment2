# web_backend_assignment2


## Objective
This project demonstrates server-side API integration. The backend fetches real-time weather data from OpenWeather and returns a cleaned JSON response. Additionally, the backend integrates a second API (REST Countries) to display extra country details.

## Tech Stack
- Node.js
- Express
- dotenv
- cors
- nodemon (dev)
